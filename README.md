# Content Manager

Content Manager é um sistema desenvolvido para auxiliar criadores de conteúdo digital na organização e gerenciamento de suas atividades.

O projeto está sendo desenvolvido com foco em oferecer uma plataforma simples e intuitiva para gerenciar tarefas, planejamento de conteúdos e produtividade.

## Tecnologias

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Maven
* PostgreSQL

### Frontend

* React *(em desenvolvimento)*

## Status do Projeto

🚧 Em desenvolvimento.

As próximas etapas incluem a implementação de uma interface web utilizando React para proporcionar uma experiência moderna e responsiva aos usuários.
